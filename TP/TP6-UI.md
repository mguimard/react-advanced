# TP6 : Bibliothère GUI

Le but de ce TP est d'utiliser la librairie graphique ant design.

## Reprise du TP précédent

Utiliser les composants suivants pour mettre un peu de style :

- Layout pour la structure principale https://ant.design/components/layout
- Button pour le bouton de login https://ant.design/components/button
- Select pour la gestion des status des employés https://ant.design/components/select
- Switch pour l'état des LEDs (enabled/disabled) https://ant.design/components/switch
- Avatar pour l'affichage des LEDs https://ant.design/components/avatar

## Bonus : fenetre de login

Utiliser Modal et Form pour un formulaire de login

- https://ant.design/components/modal
- https://ant.design/components/form#form-demo-login
